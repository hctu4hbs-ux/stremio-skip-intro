How the proxy works

The player never receives the intro — it is removed at the network level.

// Your Stremio add-on returns this stream URL:
/proxy/hls?url=<base64url_m3u8>&videoId=tt0944947:1:1

// stremio-skip-intro fetches the original manifest
// queries all sources for skip segments
// drops segments inside intro/outro ranges
// returns a clean manifest to Stremio's player

✓ Player loads. Intro never plays. No button needed.

Get started in 4 steps

From clone to working Stremio add-on in under 5 minutes.

1

Clone and install

git clone https://github.com/hctu4hbs-ux/stremio-skip-intro && npm install

2

Add your first timestamp

POST to /api/segments with the IMDB ID, episode, start and end time in seconds.

3

Wrap your stream through the proxy

Pass the original .m3u8 URL (base64url encoded) and videoId to /proxy/hls. Return the proxy URL from your stream handler.

4

Install in Stremio

Paste http://your-server/manifest.json into the Stremio Add-ons search bar and click Install.
